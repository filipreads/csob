# Finance Dashboard

## Run locally

```bash
pip install -r requirements.txt
streamlit run app.py
```

## Deploy to GitHub

1. Create a new GitHub repository.
2. Upload `app.py`, `requirements.txt`, and `README.md`.
3. Commit and push.
4. Connect the repo to Streamlit Cloud, Railway, or Render.

## Notes

Place `normalized.xlsx` in the repo root, or modify `app.py` to point to your uploaded file.
